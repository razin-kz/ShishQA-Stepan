Любой сайт в интернете находится на каком-либо устройстве, у этого устройства всегда есть свой адрес (IP-адрес). DNS - система, которая обеспечивает связь между наименованием сайта (его именем, например vk.com) и его IP адресом.
HTTP - протокол, использующийся для передачи данных в интернете посредством браузера.
HTTPS - тот же протокол, но поддерживающий шифрование, соответственно - безопасный (secure) и передаваемые данные по этому протоколу - защищены.
